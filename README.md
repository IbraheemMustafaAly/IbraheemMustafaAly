### 🦠 Gut Microbiome of *Spodoptera littoralis* *(MSc Collaboration — Dr. Gehad)*
> **Stack:** QIIME2 · Galaxy · R (phyloseq · vegan · DESeq2) · Python (NetworkX) · Tax4Fun2 · LEfSe · Random Forest

16S rRNA amplicon sequencing study across two biological subsets:
- **Larval Growth Performance:** Slow vs. Fast growers
- **Developmental Stages:** 3rd, 4th, and 5th instars

Full analytical pipeline:
- QIIME2 feature table processing and taxonomic profiling
- Alpha diversity (Shannon, Chao1) & Beta diversity (PCoA, PERMANOVA)
- DESeq2 differential abundance + LEfSe biomarker discovery
- Co-occurrence network analysis (NetworkX)
- Functional prediction with Tax4Fun2
- Random Forest classification of microbial communities
- Delivered: Colab notebooks · R scripts · full Word report with embedded figures
