### ✅ Completed

#### 🧬 Clinical Genomics Pipeline — WES Variant Calling
> **Stack:** Ubuntu 24.04 · conda · BWA-MEM · SAMtools · GATK4 · Franklin by Genoox
> 🔗 [IbraheemMustafaAly/Clinical-Genomics-Pipeline](https://github.com/IbraheemMustafaAly/Clinical-Genomics-Pipeline)

Full clinical-grade WES pipeline built from scratch:

`Raw FASTQ` → `Trimming` → `Alignment (BWA-MEM)` → `BAM Processing (SAMtools)` → `Variant Calling (GATK HaplotypeCaller)` → `VCF Filtering` → `Clinical Annotation (Franklin)`

- Built under real resource constraints (i5-4200M · 6GB RAM · Ubuntu 24.04)
- End-to-end variant interpretation with ACMG classification criteria
- Covers ClinVar, gnomAD, population frequency filtering

<br/>

#### 🧠 Parkinson's Disease Multi-Omics Data Analysis

Integrative multi-omics analysis for Parkinson's Disease biomarker discovery:
